Tiaweb encrypt
Kata sandi melindungi halaman HTML statis

Berdasarkan StatiCrypt menggunakan AES-256 untuk mengenkripsi string Anda dengan frasa sandi Anda di browser Anda (sisi klien).
Unduh string terenkripsi Anda di halaman HTML dengan prompt kata sandi yang dapat Anda unggah di mana saja (see Demo).


BAGAIMANA CARA KERJA (Tiaweb encrypt) ►

Disclaimer jika Anda memiliki data dengan ekstra sensitif, Anda mungkin harus menggunakan sesuatu yang lain!


Tiaweb encrypt menghasilkan halaman statis, kata sandi terlindungi yang dapat didekripsi di browser: cukup kirim atau unggah halaman yang dihasilkan ke tempat yang menyajikan konten statis (misalnya halaman Html) dan Anda selesai: javascript akan meminta pengguna untuk kata sandi, dekripsi halaman dan muat HTML Anda.


Ini pada dasarnya mengenkripsi halaman Anda dan menempatkan segala sesuatu dengan cara yang mudah digunakan untuk menggunakan kata sandi di file baru.

AES-256 canggih, tetapi serangan brute-force / dictionary akan sepele untuk dilakukan dengan sangat cepat: gunakan frasa sandi yang panjang dan tidak biasa!
